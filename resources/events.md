## How to Add Events

Use this format:

```markdown
### [Event Name]

- **Date:** [Month Day, Year]
- **Location:** [City, Country / Online]
- **Type:** [Hackathon / Conference / Meetup / Workshop]
- **For:** [Beginners / Experienced / All levels]
- **Topics:** [List 2-3 main topics]
- **Link:** [Registration URL]

---
```
