## Interview Preparation Strategy

### Product-Based Companies

```
Timeline: 3-6 months preparation

Month 1-2: DSA Fundamentals
├── Arrays, Strings, Hash Maps
├── Trees, Graphs, Recursion
└── Practice 100+ LeetCode problems

Month 3-4: System Design
├── Scalability concepts
├── Database design
└── Practice common designs (URL shortener, Twitter, etc.)

Month 5-6: Company-Specific Prep
├── Read company engineering blogs
├── Understand their tech stack
└── Practice behavioral questions
```

### Service-Based Companies

```
Timeline: 1-2 months preparation

Week 1-2: Core Concepts
├── OOP (Classes, Inheritance, Polymorphism)
├── DBMS (SQL queries, Normalization)
└── Basic DSA (Arrays, Strings, Sorting)

Week 3-4: Practice
├── Previous year questions
├── Aptitude and verbal reasoning
└── Mock interviews

Final Week: Company-Specific
├── Company values and culture
├── HR questions preparation
└── Resume review
```
